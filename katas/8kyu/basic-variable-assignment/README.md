## 📌 Descripción
El objetivo de esta kata es almacenar la cadena `"codewa.rs"` dentro de una variable llamada `name`.

La dificultad es mínima (8kyu) y está orientada a reforzar conceptos básicos de:

---

## 🎯 Objetivo

Construir el valor final `"codewa.rs"` a partir de dos partes:

- `"code"`
- `"wa.rs"`

Y almacenarlo correctamente en una variable llamada `name`.

---

## 📥 Entrada

Esta kata **no recibe parámetros**.

---

## 📤 Salida esperada

Una variable llamada `name` que contenga exactamente:

```text
codewa.rs
```

---

## 🧠 Conceptos trabajados

- Declaración de variables (`const`, `let` o `var`)
- Concatenación de cadenas con el operador `+`
- Asignación correcta de valores
- Precisión en nombres de variables

---

## 🗂️ Estructura del ejercicio

```text
basic-variable-assignment/
  js/
    solutions.js
  ts/
    solutions.ts
  tests/
    kata.test.js
```

- `solutions.js` → implementación en JavaScript
- `solutions.ts` → implementación en TypeScript
- `kata.test.js` → pruebas automatizadas del comportamiento esperado

---

## 🧪 Verificación

El comportamiento correcto se valida mediante tests automatizados.

La condición principal que debe cumplirse es:

- La variable `name` debe contener exactamente `"codewa.rs"`.
